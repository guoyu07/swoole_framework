swoole_framework
================

swoole framework框架文档

`./doc` 文档说明
`./src` 文档配套代码

#安装swoole拓展
`pecl install swoole`

修改`php.ini`加入`extension=swoole.so`
可参考：`https://github.com/swoole/framework/issues/6`

#下载framework框架
framework 另从github clone
`git clone https://github.com/swoole/framework.git`
